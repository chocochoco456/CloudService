<원격 리포지토리>
- 깃허브 계정 생성
- 로컬 리포지토리랑 원격 리포지토리 동기화 (업로드)

- git branch -M main (마스터 -> 메인 브랜치로 변경) (master 없을 시 git checkout -)
- git remote add origin https://github.com/chocochoco456/CloudService.git 

*local <-> remote(원격)
* origin(별명)

- git remote -v (등록 확인) (지금 이 폴더가 어디(URL)랑 연결되어 있는지 별명과 함께 보여줘!/내가 지금 이 폴더를 어느 깃허브 주소랑 연결해놨더라?)
- git push -u origin main (원격 리포지토리에 업로드 및 동기화)
- git push (u 옵션 때문에 한번만 git push -u origin main 해주면 그 다음부터는 git push만)
