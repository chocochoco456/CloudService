<버전 관리>

- git add 파일명 (스테이지에 올림(장전 상태))
- git add . (현재 폴더의 모든 파일 스테이지에 올림)
- git commit (파일명(선택)) -m "커밋 메시지" (스테이지에 올린 파일들을 커밋 기록. 해시코드가 부여되고 커밋한 아이디/이메일 등 남김(탕! 발사))
- git status (폴더 상태 감지 후 보고)
- Enter Key (복사)
- 방향키 위쪽 이동 (명령어 바로가기)
- git log (커밋 보기 확인(업데이트)/커밋 해시코드 확인)
- git checkout 해시코드 7자리(or전체) (커밋으로 시간여행(세이브포인트))
- git config --global user.email "사용자 이메일 주소" (깃 사용자 아이디 등록)
- git config --global user.name "사용자 필명" (깃 사용자 필명 등록)
